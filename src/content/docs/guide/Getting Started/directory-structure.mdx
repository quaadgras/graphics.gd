Here's the minimal `main.go` that you need to successfully launch a project:

```go
package main

import "graphics.gd/startup"

func main() {
	startup.Scene()
}
```

Start with a `main.go` file (or a few files) and gradually add `internal` packages as needed. 
Here's the recommended directory structure to follow for a `graphics.gd` project:

<FileTree>
- project_name/
    - graphics
        - project.godot
        - main.tscn
        - library.gdextension
        - .gitignore
        - export_presets.cfg
        - asset_category1
            - myimage.png
            - myshader.shader
        - asset_category2
            - myfont.ttf
            - myaudio.ogg
        - asset_category3
            - myscene.tscn
            - myscript.gd
    - internal
        - mypackage1
            - mypackage1.go
            - mypackage1_test.go
        - mypackage2
            - otherlogic.go
            - mypackage2.go
    - releases
        - windows
            - amd64/
            - arm64/
        - linux
            - amd64/
            - arm64/
        - macos
            - universal/
    - .gitignore
    - go.mod
    - go.sum
    - main.go
</FileTree>
